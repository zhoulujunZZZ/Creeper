# Creeper爬行者  
## BLDC/PMSM无刷电机驱动器  
![Creeper](https://github.com/ChenJin0927/Creeper/blob/main/Image/CREEPER.PNG)  
## 设计参考  
* 斯坦福大学四足机器人Doggo-ODrive  
* 立创EDA-ART-Pi  
## 功能  
* 通过上位机对单个电机进行CAN总线ID设置  
* 通过上位机对所有电机进行CAN总线快速ID设置  
* 针对市面上绝大多数直流无刷电机进行PID自适应调参  
* Fn功能拨码可对电机控制方式进行选择  
* 指示灯可指示当前电调工作状态  
* 其余功能详见Creeper_Readme.pdf...
